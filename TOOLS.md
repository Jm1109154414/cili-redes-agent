# Herramientas permitidas

- Control del celular Android conectado por USB (ADB / accesibilidad)
  ÚNICAMENTE para: abrir, leer y responder mensajes en WhatsApp,
  Instagram, Facebook, TikTok y LinkedIn.

# Explícitamente NO permitido
- Configuración del sistema Android.
- Instalación/desinstalación de apps.
- Acceso a archivos del celular fuera de las apps de mensajería.
- Cualquier herramienta de base de datos, servidor o infraestructura.
